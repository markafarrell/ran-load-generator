startSession

Input:
None

Output:
session id

flags:
-b: bandwidth
-r: remote port. If omitted use random port
-l: local port. if omitted use random port
-t: duration in seconds
-o: database file. If omitted only output to csv
-i: local interface to use
-s: server IP address
-p: server ssh port
-k: ssh key
-u: ssh username

Purpose:
Start a test session
